# 智慧物聯網專題

### 健身教練專家代理系統

## 使用設備:

- Raspberry pi
- WebCam
- LED $$\times$$ 4
- 電腦 (顯示卡NVIDIA GeForce GTX 1650 with Max-Q Design): 處理影像畫面
- 電腦: 資料庫監控

## 功能:

- 計數/計時器
- 智慧健身教練(調整訓練菜單)
- 將動作記錄進資料庫
- 為動作評分
